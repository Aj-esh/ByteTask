# ByteTask

ByteTask is a full‑stack application repository. The current structure includes a frontend client and an additional application workspace.

## Project Structure

```
.
├─ README.md              
├─ client/                 # Frontend (React-based)
│  ├─ README.md
│  ├─ package.json
│  ├─ package-lock.json
│  ├─ public/
│  └─ src/
└─ mern-app/               # app workspace server)
```

## Prerequisites

- Node.js
- npm
